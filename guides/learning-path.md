# 📈 Recommended Learning Path

Follow this order for the most effective CTF preparation journey.

---

## Phase 1 — Foundations (Weeks 1–3)

### Week 1: Linux Basics
- [ ] Complete [Linux Journey](https://linuxjourney.com) (first 4 modules)
- [ ] Play [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) levels 0–10
- [ ] Learn basic bash scripting

### Week 2: Networking Basics
- [ ] Study OSI model and TCP/IP
- [ ] Learn about DNS, HTTP, FTP
- [ ] Install and explore Wireshark
- [ ] Try basic nmap scans

### Week 3: Python Basics
- [ ] Complete first 10 chapters of [Automate the Boring Stuff](https://automatetheboringstuff.com)
- [ ] Write a simple script to encode/decode Base64
- [ ] Learn to make HTTP requests with `requests` library

---

## Phase 2 — Core CTF Skills (Weeks 4–8)

### Week 4–5: Web Security
- [ ] Start [PortSwigger Academy](https://portswigger.net/web-security) (SQL Injection, XSS)
- [ ] Set up Burp Suite
- [ ] Solve PicoCTF web challenges

### Week 6: Cryptography
- [ ] Complete [CryptoHack](https://cryptohack.org) introduction
- [ ] Practice with CyberChef
- [ ] Solve PicoCTF crypto challenges

### Week 7: OSINT
- [ ] Try TryHackMe OSINT rooms
- [ ] Practice Google Dorking
- [ ] Learn exiftool and Sherlock

### Week 8: Digital Forensics
- [ ] Analyze PCAP files in Wireshark
- [ ] Try steganography challenges
- [ ] Use binwalk on sample files

---

## Phase 3 — Advanced Topics (Weeks 9–12)

### Week 9–10: Reverse Engineering
- [ ] Install Ghidra
- [ ] Complete PicoCTF RE challenges
- [ ] Practice on [crackmes.one](https://crackmes.one)

### Week 11: Blockchain
- [ ] Complete first 5 Ethernaut levels
- [ ] Learn basic Solidity

### Week 12: Mixed CTF Practice
- [ ] Join a beginner CTF event
- [ ] Solve 10+ mixed challenges on PicoCTF
- [ ] Review and write up solutions

---

## 🏆 Ongoing Practice

- Participate in CTF competitions regularly (CTFtime.org lists them)
- Write writeups after every challenge — it reinforces learning
- Join CTF Discord communities
