# Learning Path

A structured 12-week plan to take you from no experience to solving beginner CTF challenges.

---

## Phase 1 — Foundations (Weeks 1–3)

### Week 1: Linux Basics
- Complete the first four modules of [Linux Journey](https://linuxjourney.com)
- Complete [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) levels 0 through 10
- Learn basic bash scripting: loops, conditionals, and file operations

### Week 2: Networking Basics
- Study the OSI model and TCP/IP stack
- Learn how DNS, HTTP, and FTP work
- Install Wireshark and capture your own traffic
- Run basic nmap scans on your local network

### Week 3: Python Basics
- Complete the first 10 chapters of [Automate the Boring Stuff](https://automatetheboringstuff.com)
- Write a script to encode and decode Base64
- Learn to make HTTP requests using the `requests` library

---

## Phase 2 — Core CTF Skills (Weeks 4–8)

### Week 4–5: Web Security
- Start [PortSwigger Web Security Academy](https://portswigger.net/web-security) — focus on SQL Injection and XSS
- Set up Burp Suite and intercept browser traffic
- Solve beginner web challenges on PicoCTF

### Week 6: Cryptography
- Complete the introduction track on [CryptoHack](https://cryptohack.org)
- Practice encoding and decoding with CyberChef
- Solve beginner crypto challenges on PicoCTF

### Week 7: OSINT
- Complete two OSINT rooms on TryHackMe
- Practice Google dorking using the Google Hacking Database
- Try extracting metadata with exiftool and running Sherlock

### Week 8: Digital Forensics
- Analyze a PCAP file in Wireshark
- Solve a steganography challenge using steghide or zsteg
- Run binwalk on a sample firmware file

---

## Phase 3 — Advanced Topics (Weeks 9–12)

### Week 9–10: Reverse Engineering
- Install Ghidra and analyze a simple binary
- Complete PicoCTF reverse engineering challenges
- Practice on [crackmes.one](https://crackmes.one)

### Week 11: Blockchain Security
- Complete the first five levels of Ethernaut
- Learn basic Solidity syntax and contract structure

### Week 12: Mixed Practice
- Participate in a beginner-friendly CTF event (see [CTFtime](https://ctftime.org))
- Solve at least 10 mixed challenges on PicoCTF
- Write up at least two solutions to reinforce your learning

---

## Ongoing Practice

- Participate in CTF competitions regularly — CTFtime.org lists all upcoming events
- Write solution writeups after every challenge
- Join CTF Discord communities and ask questions
